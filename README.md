## ⚙️ Features

- Secure login system for admin and employees
- Create, assign, and manage tasks
- View task stats: new, active, completed, failed
- Clean dark-themed UI

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- React.js
- Local Storage (for database)
- Netlify (for live hosting)
